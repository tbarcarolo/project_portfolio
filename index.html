Análise de Dados do Airbnb com Python

Desenvolvi uma análise dos dados de anúncios do Airbnb, com o objetivo de extrair insights sobre imóveis, anfitriões e avaliações. Utilizei Python, Pandas e Matplotlib para realizar análises estatísticas, gerar gráficos e criar um menu interativo no terminal. O projeto resultou em uma ferramenta que facilita a visualização dos dados e a identificação de padrões importantes.

As ferramentas utilizadas foram:

  • Python — Linguagem de programação principal do projeto.
  • Pandas — Para manipulação, limpeza e análise dos dados.
  • Matplotlib — Para construção de gráficos e visualizações.
  • Jupyter Notebook — Para desenvolvimento da análise exploratória.
  • Script Python Interativo (Terminal) — Criação de um menu interativo para navegação pelas análises.
  • CSV Dataset (airbnb_listings.csv) — Base de dados utilizada para as análises.

Análise e Visualização de Dados com Python e Pandas

Durante meus estudos em análise de dados, meu desafio foi explorar três datasets públicos: College, Auto e Boston Housing. Apliquei técnicas de limpeza, criação de variáveis categóricas, análise estatística e visualização com Pandas, Matplotlib e Seaborn. Com isso, consegui gerar insights sobre custos universitários, consumo de veículos e fatores que impactam o valor dos imóveis e a criminalidade em Boston.

As ferramentas utilizadas foram:

  • Python — Linguagem principal para desenvolvimento da análise.
  • Pandas — Manipulação, limpeza e análise dos dados.
  • Numpy — Cálculos estatísticos e operações numéricas.
  • Matplotlib — Criação de gráficos e visualizações.
  • Seaborn — Visualizações estatísticas avançadas.
  • Jupyter Notebook — Ambiente utilizado para desenvolvimento e execução da análise exploratória.
  • CSV Dataset (College.csv, Auto.csv, Boston.csv) — Bases de dados utilizadas para as análises.

Banco de dados relacional sobre animes em SQL com MySQL Workbench

Criei um banco de dados relacional sobre animes para praticar habilidades em SQL, modelagem de dados e relacionamento entre tabelas. O projeto foi estruturado diretamente no MySQL, onde desenvolvi as tabelas, populando-as com dados relevantes e criando relações por meio de chaves primárias e estrangeiras. Posteriormente, gerei o diagrama ER via engenharia reversa no MySQL Workbench para ilustrar a estrutura do banco. Esse projeto simula um cenário real de análise e estruturação de dados com base em múltiplas entidades conectadas.

As ferramentas utilizadas foram:

  • MySQL — Sistema de gerenciamento de banco de dados utilizado em todo o projeto.
  • SQL — Linguagem usada para criação das tabelas, relacionamentos e consultas.
  • MySQL Workbench — Ambiente de desenvolvimento e modelagem, também utilizado para gerar o diagrama ER por engenharia reversa.
  • CSV Dataset (data_anime.csv) — Base de dados usada para popular as tabelas do banco.
